# Markflow 支持 / Support

Markflow 是一款简洁的 Markdown 笔记应用,支持目录管理、模板库、版本历史、ZIP / Jupyter Notebook 导入、PDF 与长图导出。

Markflow is a clean Markdown note-taking app with folders, a template library, version history, ZIP / Jupyter Notebook import, and PDF / long-image export.

## 常见问题 / FAQ

**高级版是订阅吗? / Is Premium a subscription?**

不是。高级版是一次性买断,永久有效。 / No — it's a one-time purchase, yours forever.

**换了设备怎么恢复高级版? / How do I restore Premium on a new device?**

在「设置 → 高级版」页面点「恢复购买」。 / Open *Settings → Premium* and tap *Restore purchase*.

**如何删除账户? / How do I delete my account?**

在「设置 → 账户 → 删除账户」中操作,账户和云端数据会立即永久删除。 / Use *Settings → Account → Delete account*; your account and synced data are deleted immediately and permanently.

**不登录能用吗? / Can I use the app without an account?**

可以,登录页选择「暂不登录,直接使用」,笔记保存在设备本地。 / Yes — tap *Skip for now* on the login page; notes are stored locally on your device.
